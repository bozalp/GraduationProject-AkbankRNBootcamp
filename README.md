# Graduation Project-Akbank React Native Bootcamp
<br>
This project designed with Expo.<br><br>

<b>Packages used</b><br>
-React Navigation<br>
-React Native Icons<br>
-React Native Picker<br>
-AsyncStorage<br>
-Redux-Toolkit<br>
-Firebase<br>

Includes dark and light theme.
